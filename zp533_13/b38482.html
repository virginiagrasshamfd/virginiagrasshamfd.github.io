- Features -
Function 01
ウッドフレーム

ボールドなウッドフレームで囲ったスクエアのフォルムが印象的。上質な木の質感と角を丸めた仕上げがマッチしたインテリアアイテムです。
   
Function 02
コントラスト

幅広の長針と短針に合わせたのは背の低いフォントの数字と放射状に伸びるインデックス。秒針にはアクセサリーのような華奢なデザインを合わせてユニークなコントラストを演出。
   
Function 03
視認性

スイープムーブメント

カチコチ音がしない連続秒針ムーブメントを使用しているので、寝室などの静かなお部屋にもおすすめです。
 
Function 05
サイズ感

厚みのあるウッドフレームに対して僅かに小ぶりなサイズ感ながら、視認性や存在感はしっかり確保。

- Brand information -
 
「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
