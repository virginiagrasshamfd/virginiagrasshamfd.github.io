透明なガラスに真鍮製のソケットを合わせ、片手に収まるほどのコンパクトさを追求しました。ガラスの厚みは残しつつ、ぎゅっと小さくしました。クリアガラスを通して、揺らぎのあるピュアな光が楽しめます。複数でも並べやすいサイズながら、1灯でもしっかりとした存在感を放ちます。

- Features -
Function 01
真鍮

ソケットは贅沢な真鍮製。鈍いゴールドの輝きと経年変化をお楽しみいただけます。
Function 02
たまり

内側にあえてヨレやたまりを残すことで、独特な揺らぎが生まれます。ハンドメイドのていねいな手仕事が感じられます。
Function 03
コンパクト

玄関やトイレはもちろん、廊下やキッチンなどでは複数でお楽しみいただけるコンパクトなサイズにこだわりました。
Function 04
レトロ

ブラウンカラーのツイストコードとゴールドのソケットデザインがレトロな印象を与えています。
Function 05
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

- Brand information -

「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、
みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
