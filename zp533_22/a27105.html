- Features -
■ Function 01
球体美

シンプルで飽きの来ない球状のフォルム。存在感を示し過ぎず様々なインテリアシーンにおいて、緩やかにマッチングする。宙に浮かぶ球体のオブジェのような佇まいで灯りをもたらします。
   
■ Function 02
連続性

1灯吊りだけでなく2個3個と並べてのライティングもお部屋に映えます。高さを均等したり、逆に不均等にしたり。サイズ違いのSession-dining pendantと組み合わせたり可能性は無限。同じ球体が続く様は美的知覚を刺激します。
   
■ Function 03
穏やかに透ける灯り

シェードはやや内部の光を透過する素材、加工になっています。ほんのりと球体自体が光っているように、柔らかい光が広がり、お部屋の雰囲気を一層印象的に。
   
■ Function 04
僅かな演出

シェード上部に僅かな隙間を作っています。ここから漏れ出る灯りが少しばかり上部を照らします。ちょっとした演出が心憎い仕様になっています。
