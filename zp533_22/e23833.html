スキンケアの基本「汚れを落とす」と「肌を保湿する」がこれ一台でケアでき、肌本来のキレイを効果的に引き出します。簡単だから続けられる、普段のスキンケアを格上げます。ナノサイズの温かいスチームに包まれるエステのような心地よさで、毛穴を開き普段のクレンジングでは落としきれない汚れを落とします。温かいスチームで保湿されうるおった肌は、キメが整いメイクのノリも UP します。

- Features -

Function 01
100 万分の1

1mm の 100 万分の1のナノミストで層違いのクレンジング。数あるスチーマーの中でも、ミストのきさには差があります。ナノミストはミストの中でも特にさい、ナノサイズ(1/1,000,000mm)であることが実証されたミストです。

Function 02
ディープクレンジング

回転しながら噴射するミストを独自開発。スパイラル噴射でミストが顔を包み込み、密着。角質層まで浸透してしっかり汚れを浮き上がらせます。

Function 03
実証された、高いクレンジング

実は、通常のクレンジングでは落としきれない汚れも。ミストの小さい、フェイシャル クレンジング ナノスチーマーをプラスすることで質層の汚れまでケアすることができます。

Function 04
肌の奥までたっぷり保湿

ナノサイズのミストだから、なにもつけていない肌にスチーマーをあてるだけで水分量が大幅にUP。約45℃の温かいスチームで角質層まで水分を届け、肌をやわらかくするので、スチーム後のスキンケアと合わせればさらなる効果が期待できます。

Function 05
スキンケアをバックアップ

ケア中の肌を確認できるガラス製のミラーとLEDライトを装備。明るくて見やすいミラーは肌のコンディションチェックに使用可能。操作もシンプルで煩わしさ等がありません。
Function 05
サイズ感

コンパクトで軽量。出しておいてもかわいいコロンとしたデザイン。使いたいときにサッと使える手軽さ。
