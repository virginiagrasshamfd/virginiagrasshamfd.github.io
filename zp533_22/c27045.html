ミニマルモダンなシルエットに素材のコントラストを効かせた、トレンドライクなペンダントライト。揺らぎのあるガラスに大理石やトラバーチンを合わせた、今の気分にフィットする仕上がりです。ダイニングやキッチンでの多灯使いも楽しめるコンパクトなサイズです。

- Features -
Function 01
ミニマル

モダンな雰囲気を残したまま、ミニマルさを追求したデザインです。素材やカラーリングで表情豊かな装いに仕上げました。
Function 02
揺らぎ

小さなガラスセードをよく見ると水面のような揺らぎが見られます。向こう側が歪んで見える様が繊細で情緒的な印象を与えます。
Function 03
コントラスト

重厚感のある大理石とガラスが生むハイコントラストが、今の気分にフィットする組み合わせです。
Function 04
アクセント

アクセントになっているガラスホルダーは、クリアにはゴールドカラーを、ピンクにはコッパーカラーを採用しています。
Function 05
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
